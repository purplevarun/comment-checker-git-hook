## INSTALLATION 💪

You can copy paste the below snippet in the home directory of your repository!

<pre>

curl https://raw.githubusercontent.com/purplevarun/comment-checker-git-hook/main/install.sh | bash
rm -rf hook-creator.cc
</pre>
